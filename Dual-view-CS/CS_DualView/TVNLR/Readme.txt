This demo implements the stategy for Block based Image Compressive Sensing of Paper "Improved Total Variation based Image Compressive Sensing Recovery by Nonlocal Regularization".

-------------------------------------------------------------------------------------------------------
The TVNLR software implemented by MatLab 7.10.0(2010a) are included in this package.

Run the Demo.m file directly.
